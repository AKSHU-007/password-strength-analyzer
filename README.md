# 🔒 Password Guardian AI

An AI-powered cybersecurity web application that analyzes password strength and provides real-time security recommendations.

## 🌐 Live Demo

[https://YOUR-RENDER-LINK.onrender.com](https://password-strength-analyzer-zllu.onrender.com)

## ✨ Features

✅ Real-time password strength analysis  
✅ Percentage-based security score  
✅ AI recommendations for stronger passwords  
✅ Animated security dashboard  
✅ Interactive modern UI  
✅ Cybersecurity-focused design
